## Running the application locally

clone the rep
```
$ git clone <repo-url>
```

create a virtual environment
```
$ python -m venv venv
```

activate the virtual environment

linux: `$source venv/bin/activate`

windows: `$venv/Scripts/activate `

run the application

`$ flask run`
